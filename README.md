# Task: ReactJS - Hello World

**Author:** D P Rithvik Kumar

## Description & Learning Process
This is my very first React application! The prompt allowed the use of either `create-react-app` or Vite. I chose to use Vite because the React documentation now recommends modern bundlers, and I noticed it installs and starts up significantly faster than CRA.

To build this, I:
1. Scaffolded the project using the Vite React template.
2. Navigated into the `src/App.jsx` file and stripped out all the default boilerplate code (like the counter and logos).
3. Created a clean functional component that simply returns an `<h1>` tag with "Hello, World!" inside.
4. Added a bit of inline CSS styling using a JavaScript object to center the text perfectly on the screen using Flexbox.

## How to Run
1. Open your terminal in this project folder.
2. Run `npm install` to download the required dependencies.
3. Run `npm run dev` to start the local development server.
4. Open the provided `localhost` link in your browser to see the app.
